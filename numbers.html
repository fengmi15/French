Compound Numbers and Tens - Les Nombres Composés et les Dizaines

Pattern for 21-29:

For 21-29, use "vingt" + hyphen + basic number (21 = vingt-et-un, 22 = vingt-deux, 23 = vingt-trois, etc.)

Special French Number Rules:

70s: Use "soixante" (60) + number (70 = soixante-dix, 75 = soixante-quinze)
80s: Use "quatre-vingts" (four twenties) but drop 's' when followed by a number
90s: Use "quatre-vingt-dix" + number (92 = quatre-vingt-douze)

Million: Takes 's' in plural (deux millions)

Billion: French billion = English trillion (10¹²)
Important: French and English billion systems differ!

Key Rules Summary:

1. Use hyphens for compound numbers under 100
2. "Et" only appears in: 21, 31, 41, 51, 61, 71
3. No "et" in: 81, 91, 101, etc.
4. "Cent" and "vingt" take 's' only when they end the number
5. "Mille" never takes 's'
